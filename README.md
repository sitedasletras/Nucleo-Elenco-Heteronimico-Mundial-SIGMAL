# Núcleo de Elenco Heteronímico Mundial SIGMAL

Projeto individual do eixo Site das Letras / SIGMAL para organização de elenco audiovisual heteronímico permanente.

Este núcleo é independente da OHE, do Lapidar e do Caminho das Pedras.

## Finalidade

Criar e organizar atores, atrizes, narradores, apresentadores e personagens para produções da Web TV Celeiro, Web Rádio Celeiro e Silos SIGMAL.

- docs/

## Regra de continuidade

Cada integrante deve manter nome artístico, rosto reconhecível, biotipo-base, território estético e estilo de atuação.

Podem variar figurino, cabelo, barba, maquiagem, iluminação, cenário e leve variação de peso.
